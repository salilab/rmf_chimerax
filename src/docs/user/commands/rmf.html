Command: rmf

Usage:
rmf chains  model

Usage:
rmf hierarchy  model [ depth N ]

Usage:
rmf readtraj  model [ first N ] [ last N ] [ step N ]

The rmf chains command, given a model previously read from an RMF file, shows the RMF name for each chain in the model.

The rmf hierarchy command, given a model previously read from an RMF file, displays the RMF hierarchy in the log. depth if given limits the print to the N levels under the root of the hierarchy - for example a depth of 0 will show only the root node, 1 will show the root and only its immediate children, and so on.

The rmf readtraj command, given a model (an RMF state previously read from an RMF file), will load all frames from the file. These can then be displayed using the coordset slider command.

By default all frames are read from the file. This can be controlled by setting the first frame to read (default 0) and/or the last frame (default the last frame in the file) and/or the step (default 1).

Note that this command reopens the existing RMF file, so it will likely fail if the RMF file has been modified externally since it was originally opened in ChimeraX.
