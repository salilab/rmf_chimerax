Tool: RMF Viewer

The RMF Viewer tool displays information contained in models read from RMF files. (Note that currently only information on the first such model is shown.)

The Hierarchy frame shows the hierarchy of nodes in the RMF file. One or more nodes can be selected and the corresponding structure selected, shown, or hidden using the buttons on the right side.

Sali Lab, University of California San Francisco / February 2020
